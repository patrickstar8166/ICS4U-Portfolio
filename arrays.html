Arrays

What are Arrays?

Arrays are a block of consecutive memory locations that hold values of the same data type, with the individual locations being called elements. Rather than treating each element as a separate variable, the whole array gets one name, which makes code more efficient if there are multiple similar variables (ex. student1, student2...).

The specific elements in an array are referred to by its index, which is an int value from 0 to length-1. Arrays have fixed size and thus their length cannot be changed. Note that the Java interpreter checks the values of indices at runtime and throws ArrayIndexOutOfBoundsException if an index is not possible to access.


Parallel Arrays and Syntax

Before use, arrays must be declared. The syntax used is anyType[] arrName; Declaration only creates a reference set to null, and thus arrays must be initialized before use. Arrays are objects, and are thus declared with the new keyword.


Different ways of initialization:

  • No values: anyType[] arrName = new anyType[length];
  • Preset values: anyType[] arrName = {element, element, ...};
When arrays are initialized without values, all elements are set to default values of either 0 for numbers, false for booleans, or null for objects. The length of array is either given explicitly or comes from the length of the {...} initialization list. To access the length of an array, use arrName.length, which is a public field in an array object

All array elements can be accessed with the syntax arr[k], indicating the k-th index. The first element is a[0] and the n-th element is a[n-1].

Example:

int[] arr1 = new int[5]; //array of size 5 with default values
double[] arr2 = {0.1, 0.2, 0.3}
System.out.print(arr2[1]); //prints 0.2



Passing arrays into methods: public void method(anyType[] arrName). The elements of the original array are not copies as with primitive data types, so they are accessible and mutable in the method’s code.

Returning arrays from methods: public anyType[] method()


2D Arrays and Syntax

2D arrays are in the form of a rectangular board. They are inherently an array of arrays. Each row is stored in a separate block of consecutive memory locations. Elements are accessed using the row then column (ex. board[r][c]). Like parallel arrays, row and column indices start at 0.


Declaration: anyType[][] arrName;
Initialization:

  • No values: arrName = new anyType[rows][columns];
  • Preset values: anyType[][] arrName = {{element, element}, {element, element}}
arrName.length is number of rows, and arrName[n].length is length of the n-th row.


Traversing Arrays

Parallel arrays:

  • Using a for loop: for (int i = 0; i < arr.length; i++) //process arr[i]
  • Using a for each loop:
    • Introduced in Java 5 and is convenient for traversing arrays
    • Syntax: for (int i : arr) - “for int i in arr”
    • Downside is that elements cannot be changed
2D arrays:
  • Using a nested for loop:
    • for (int i = 0; i < arr.length; i++){
    for (int j = 0; j < arr[i].length; j++){
        //process arr[i][j]
    }
